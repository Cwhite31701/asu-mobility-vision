```

├── train.py              # Training pipeline (auto-label + fine-tune)
├── requirements.txt      # All dependencies
├── data/
│   ├── frames/           # (auto-populated from dataset)

│   └── mobility_v1/weights/

├── logs/                 # CSV mobility logs (auto-created)

    ├── auto_label.py     # Auto-labeling with pretrained YOLOv8n
    ├── congestion.py     # Congestion scoring + prediction
    ├── overlay.py        # Drawing utilities + heatmap
    ├── stream.py         # ASU live stream capture
    └── tracker.py        # SORT multi-object tracker
```

## ⚡ Quick Start (5 minutes)

### Step 1 — Install dependencies
```bash
cd "/Users/chandlerwhite/Desktop/Demo video /mobility_vision"
pip install -r requirements.txt
```

### Step 2a — Run dashboard immediately (no training needed)
Uses pretrained YOLOv8n directly — great for a quick demo!
```bash
streamlit run dashboard.py
```
In the sidebar, select **"⚡ Pretrained YOLOv8n"** and **"📁 Demo Video"** or **"📸 Demo Frames"**, then press **▶ Start**.

### Step 2b — Train the fine-tuned model (better accuracy, ~5–15 min)
```bash
python train.py
```
This will:
1. Auto-label 216 frames using pretrained YOLOv8n
2. Fine-tune YOLOv8n for 30 epochs on person + bike_wheel classes
3. Save best model to `models/mobility_v1/weights/best.pt`

After training, restart the dashboard and select **"🏋️ Fine-tuned"** model.

## 🎮 Dashboard Guide

| Control | Description |
|---|---|
| **Video Source** | Live ASU stream, local MP4 videos, or extracted frames |
| **Model** | Fine-tuned (post-training) or pretrained YOLOv8n |
| **Confidence** | Detection threshold (0.20–0.80) |
| **SORT Tracking** | Persistent track IDs across frames |
| **Heatmap** | Density overlay of movement zones |
| **Rolling Window** | Seconds for congestion average |
| **▶ Start / ⏹ Stop** | Toggle detection loop |
| **⬇ Download CSV** | Export session mobility logs |

### Congestion Levels
| Score | Status | Meaning |
|---|---|---|
| < 5 | 🟢 LOW | Normal traffic |
| 5–12 | 🟡 MEDIUM | Moderate activity |
| > 12 | 🔴 HIGH | Alert: high congestion |

**Score Formula:** `Walkers × 1.0 + Wheeled × 1.5`

## 🔴 Live Stream Notes
The ASU Hayden webcam (`https://view.asu.edu/tempe/hayden`) uses a browser-embedded player. The app attempts to extract the HLS stream URL automatically. If extraction fails, it falls back to the local demo videos seamlessly.

## 🐛 Troubleshooting

**`ModuleNotFoundError: ultralytics`**
```bash
pip install ultralytics
```

**`ModuleNotFoundError: filterpy`**
```bash
pip install filterpy lapx
```

**Slow performance on CPU**
- Use YOLOv8n (nano) — already configured
- Lower target FPS to 2–3 in sidebar
- Disable heatmap overlay

**Training fails / out of memory**
```bash
python train.py --epochs 20
```
